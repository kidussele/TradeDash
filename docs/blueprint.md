# **App Name**: TradeDash

## Core Features:

- Dashboard Overview: Display key metrics such as Net P&L, Trade Expectancy, Profit Factor, and Win Rate.
- Interactive Charts: Visualize Daily Net Cumulative P&L and Net Daily P&L with interactive charts.
- Zella Score: Calculate and display the Zella Score, providing an overview of trading performance.
- Open Positions/Recent Trades: List open positions and recent trades with details like Open Date, Symbol, and Net P&L.
- Calendar View: Show a calendar view with trade summaries for each day, including the total P&L and number of trades.
- Import Trades: Allow users to import their trade data from various sources.
- Intelligent Insights: An AI-powered tool that provides insights on trading habits, and suggests potential improvements by identifying patterns from historical trade data.

## Style Guidelines:

- Primary color: Soft purple (#A381EB) to give a modern and trustworthy impression.
- Background color: Very light gray (#F7F5FA) to provide a clean and spacious feel.
- Accent color: Blue (#63B5FF) for interactive elements and highlights, ensuring visibility and ease of use.
- Body and headline font: 'Inter', a sans-serif font, will be used throughout the application to provide a modern, neutral and readable feel.
- Use simple, outline-style icons for navigation and key actions. Icons should be easily recognizable and consistently styled.
- Employ a clean, card-based layout with generous spacing to create a structured and easily scannable interface.
- Subtle animations on data updates and transitions to enhance the user experience without being distracting.
